## 前言

《Interpy-zh》是《Intermediate Python》的中文译本, 谨以此献给进击的 Python 和 Python 程序员们!

本仓库基于 《[eastlakeside](https://github.com/eastlakeside)/**[interpy-zh](https://github.com/eastlakeside/interpy-zh)**》 进行学习而创建的，请勿用于商业活动（仅供学习使用）。
